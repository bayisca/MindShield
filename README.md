# MindShield 🛡️

MindShield, anonim ruh sağlığı desteği ve topluluk etkileşimi için geliştirilmiş profesyonel bir JavaFX masaüstü uygulamasıdır.

## ✨ Özellikler
- **Persona Sistemi:** Gerçek kimliğinizi paylaşmadan anonim üyelik.
- **Uzman Desteği:** Onaylı danışmanlarla özel mesajlaşma.
- **SoulShield:** Dinlendirici meditasyon müzikleri.
- **JourShield:** Ruh hali takibi ve kişisel günlük.
- **Forum & Blog:** Topluluk tartışmaları ve profesyonel içerikler.
- **Admin Kontrolü:** Güvenli moderasyon ve kullanıcı yönetim paneli.

## 🚀 Kurulum ve Çalıştırma

### Gereksinimler
- **Java JDK 21** veya üzeri
- **Maven 3.8+**

### Adımlar
1. Projeyi bilgisayarınıza indirin veya klonlayın:
   ```bash
   git clone https://github.com/kullaniciadi/mindshield.git
2.Proje dizinine gidin:
 '''bash
cd MindShield

3.Maven bağımlılıklarını yükleyin ve projeyi derleyin:
 '''bash
mvn clean install

4.Uygulamayı başlatın:
 '''bash
mvn javafx:run
